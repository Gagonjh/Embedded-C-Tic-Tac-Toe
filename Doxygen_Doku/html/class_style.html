Definiert das Farbthema der Anwendung.

Die Klasse stellt unterschiedliche Farbthemen für den Nutzer zur Wahl zur verfügung. Die Klasse ist ein Singleton somit gibt es nur eine Instanz auf die man mit Style::instance(). zugreifen kann.
